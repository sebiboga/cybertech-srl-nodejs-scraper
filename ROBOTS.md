# Robots.txt Analysis — cybertech.ro

Sursa: https://cybertech.ro/robots.txt

## Reguli

```
User-agent: *
Allow: /
```

## Interpretare

Site-ul cybertech.ro permite accesul tuturor crawlerelor pe toate căile. Nu există restricții în robots.txt.

## Recomandare

Scraperul nu accesează direct site-ul cybertech.ro pentru extragerea job-urilor. Se folosește API-ul public ANOFM (Agentia Nationala pentru Ocuparea Fortei de Munca) pentru a căuta job-uri după CIF-ul companiei.
